How to Build a Hugo Blog on VPS with PHP Search Functionality

Creating a fast, scalable blog using Hugo on a VPS can be highly rewarding. Hugo is a static site generator that allows you to build a website quickly without relying on heavy backend frameworks. By hosting on your own VPS, you gain full control over performance, security, and customization. In this guide, we’ll cover the steps to set up Hugo on a VPS and integrate a PHP-based search function to handle dynamic content searches.

Step 1: Prepare Your VPS

Start by choosing a reliable VPS provider such as Ubuntu 20.04. Update your server packages:

sudo apt update && sudo apt upgrade -y
sudo apt install git curl unzip php-cli php-mbstring

This ensures you have Git, PHP, and other essential tools installed.

Step 2: Install Hugo

Download and install the desired version of Hugo. For example:

wget https://github.com/gohugoio/hugo/releases/download/v0.115.0/hugo_0.115.0_Linux-64bit.deb
sudo dpkg -i hugo_0.115.0_Linux-64bit.deb

Verify the installation:

hugo version

Step 3: Initialize Your Hugo Site

hugo new site myblog
cd myblog
git init

You can now add a theme. For a dark theme, choose one compatible with Hugo’s configuration:

git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
echo 'theme = "ananke"' >> config.toml

Step 4: Create Content

Add new posts with:

hugo new posts/my-first-post.md

Edit the markdown file under content/posts/ and fill in your content. Hugo will automatically generate static HTML pages when you build the site.

Step 5: Build and Serve Your Site

Build your site for production:

hugo --gc --minify --enableGitInfo -v

Serve the site locally for testing:

hugo server -D

Step 6: Integrate PHP Search

Hugo generates static pages, so for dynamic search, we can use PHP. Create a search.php file in your public/ folder:

<?php
$query = $_GET['q'] ?? '';
$results = [];
if ($query) {
    $files = glob('*.html'); // adjust to your content folder
    foreach ($files as $file) {
        $content = file_get_contents($file);
        if (stripos($content, $query) !== false) {
            $results[] = $file;
        }
    }
}
?>

<h2>Search Results for '<?php echo htmlspecialchars($query); ?>'</h2>
<ul>
<?php foreach ($results as $result) : ?>
    <li><a href="<?php echo $result; ?>"><?php echo $result; ?></a></li>
<?php endforeach; ?>
</ul>

This PHP script scans the generated HTML files for the query keyword and returns matching results. You can enhance it with indexing, caching, or AJAX for faster results.

Step 7: Deploy Your Site

Upload the public/ folder to your VPS web directory, e.g., /var/www/html. Make sure PHP is enabled in your web server configuration. You now have a Hugo site hosted on your VPS with dynamic PHP search.

By combining Hugo's static site speed with PHP's dynamic search capability, you can create a fast, searchable blog that scales efficiently without relying on complex CMS frameworks.
